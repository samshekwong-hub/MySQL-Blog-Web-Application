# MySQL-Blog-Web-Application
A MySQL Blog web application with using flask allowing adding or deleting blogs
The SQL script could be found in **db_backup** folder

### Steps
1. Create a database named "blog_db" and import the SQL script into the database
2. Inside the project, create a Python virtual environment
3. Open the terminal, activate the program by typing ``` python app.py ```
4. After the application run successfully, click the ``` Running on http://localhost:5000 ```
